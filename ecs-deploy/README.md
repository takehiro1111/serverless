# Ecs Deploy

nextbeat-actions を dispatch して ECS をデプロイするための Lambda

## Setup

Run this command to initialize a new project in a new working directory.

```
npm install
```

## Usage

**Deploy**

```
sls deploy --verbose
```

**Invoke the function locally.**

```
sls invoke local --function ****
```

**Remove**

```
sls remove --verbose
```
